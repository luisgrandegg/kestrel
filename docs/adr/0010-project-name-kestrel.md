# ADR-0010: Project name — Kestrel

**Status:** Accepted — 2026-07-12

## Context
The project needed a short, memorable name for the repo and docs. (Recorded for completeness; this decision is non-architectural.)

## Decision
**Kestrel** — the falcon that hovers dead still, watches, and drops only when the moment is right. It mirrors the tool's stance: a watchlist that waits and flags mechanically-defined opportunities rather than acting or advising.

## Consequences
- Ties the name to the product's stance (patient, mechanical, candidates-not-recommendations), reinforcing the framing set in the constitution.

## Alternatives considered
- **Coil** — trader shorthand for a stock compressing before a move; nodded at the volatility screen.
- **Cusp** — "on the cusp of a catalyst"; covered the earnings/ex-div screens.
- **Updraft**, **Sift** — in the same short, verb-adjacent vein as the user's other tools.
